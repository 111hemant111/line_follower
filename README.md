# Line Follower Robot
A robot based on Infrared transmitters and receivers to track a black line on white backdrop.

## Status
Completed. Scope for improvisation. Last update Mar 2016.

## Components used
1. Arduino Uno
2. Infrared sensors
3. LEDs
4. 16x2 LCD Display
5. DC Motors - 2
6. Misc. components
